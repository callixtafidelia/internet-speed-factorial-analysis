# Investigating Factors Influencing Internet Speed on UBC Vancouver Campus

## Overview

This project investigates how internet download speed varies across different conditions on the University of British Columbia (UBC) campus. A Randomized Complete Block Design (RCBD) with a multi-factor factorial structure was implemented to evaluate how campus location, building position, device type, browser, and time of day affect download speed.


## Methodology 
- Randomized Complete Block Design (RCBD)
- Linear Modeling and Factorial ANOVA
- Box–Cox Transformation  
- Tukey HSD Post-Hoc Comparisons
- Residual Diagnostics and Assumption Checking
- Interaction Modeling (Up to 3-Way Effects)

## Key Findings

- Campus Area, Building Position, and Time of Day significantly influenced download speed
- Morning speeds were significantly higher than afternoon speeds
- Indoor measurements outperformed edge and outdoor locations
- Device type and browser were not statistically significant
- The blocking factor (speed test website) was not statistically significant
